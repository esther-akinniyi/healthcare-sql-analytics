# Project Summary

## Background

Healthcare organizations generate large amounts of clinical and operational data every day. SQL is one of the primary tools used by Clinical Data Managers, Healthcare Analysts, and Business Intelligence professionals to transform raw healthcare data into meaningful insights.

This project demonstrates practical SQL techniques for exploring a simulated healthcare database.

---

## Business Questions Answered

The project answers questions including:

- How many patient encounters occurred before 2020?
- How many unique patients received care?
- What encounter types were most common?
- Which counties served the largest patient populations?
- Which emergency conditions were documented most frequently?
- What was the average emergency department throughput?
- Which insurance payers provided the highest coverage?
- Which clinical procedures were performed most frequently?
- Which patient populations experienced the highest prevalence of hypertension?

The project demonstrates the ability to write SQL queries that answer meaningful healthcare questions while supporting data-driven decision making in clinical and operational environments.
