# Healthcare SQL Analytics

## Project Overview

This project demonstrates SQL-based healthcare analytics using a simulated healthcare database. The analyses were completed as part of my transition into Clinical Data Management and Healthcare Analytics.

The project focuses on answering real-world healthcare operational and clinical questions using SQL and relational database concepts. It demonstrates experience querying healthcare datasets, performing data validation, calculating healthcare metrics, and generating analytical reports.

---

## Objectives

- Analyze healthcare encounter activity
- Explore patient demographics
- Evaluate emergency department performance
- Analyze healthcare costs and payer coverage
- Assess clinical procedure utilization
- Identify hypertension trends using blood pressure observations
- Perform financial and operational healthcare reporting
